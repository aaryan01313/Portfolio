# My Personal Portfolio

Welcome to my personal portfolio repository! This website showcases my projects, skills, and coding journey, with a primary focus on Data Science and Artificial Intelligence.

## 🚀 About the Project

This portfolio is a modern, responsive web application designed to highlight my work in Machine Learning, Algorithmic Problem Solving, and software development. It serves as a central hub for my professional identity, linking to my various coding profiles and detailing my key achievements.

### Key Features
* **Project Showcase:** Detailed overviews of my core projects, such as my End-to-End Weather Prediction Model and my finalist entry for the National CyberShield Hackathon '25.
* **Skill Matrix:** A visual representation of my technical toolkit, including Python, C++, MySQL, and essential Data Science libraries (NumPy, Pandas, Scikit-Learn, etc.).
* **Coding Profiles:** Quick links to my active profiles on LeetCode and GitHub, demonstrating my consistent problem-solving streak.
* **Responsive Design:** Fully optimized for seamless viewing across desktop, tablet, and mobile devices.

## 🛠️ Built With

This project leverages a modern frontend stack to ensure high performance and a clean user experience:

* [Next.js](https://nextjs.org/) - React Framework for production
* [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for rapid UI development
* [Framer Motion](https://www.framer.com/motion/) - For smooth animations and transitions
* [React Icons](https://react-icons.github.io/react-icons/) - For scalable vector icons

## 💻 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1.  Clone the repository
    ```bash
    git clone [https://github.com/aaryan01313/Portfolio.git](https://github.com/aaryan01313/Portfolio.git)
    ```
2.  Navigate to the project directory
    ```bash
    cd Portfolio
    ```
3.  Install NPM packages
    ```bash
    npm install
    ```
4.  Run the development server
    ```bash
    npm run dev
    ```
5.  Open your browser and visit `http://localhost:3000`
